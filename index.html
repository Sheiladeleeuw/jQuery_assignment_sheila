What is jQuery?

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT license. Web analysis indicates that it is the most widely deployed JavaScript library by a large margin. jQuery's syntax is designed to make it easier to navigate a document, select DOM elements, create animations, handle events, and develop Ajax applications.

Animate elements

The jQuery library provides several techniques for adding animation to a web page. These include simple, standard animations that are frequently used, and the ability to craft sophisticated custom effects. with handler events you can animate the different elements on your webpage.

Handler events

Events are methods that are used to register behaviors to take effect when the user interacts with the browser, and to further manipulate those registered behaviors.
An example of an event is Click(). The click event is sent to an element when the mouse pointer is on the element, and the mouse button is pressed and released. Any HTML element can receive this event. Click on the text of any of the paragraphs, the navigator bar or the main heading to see what event "click" can do together with and edded.
With the .hover() method you can binds handlers for both mouseenter and mouseleave events. You can use it to simply apply behavior to an element during the time the mouse is within the element. go over the navigator bar to see what hover does.
The contextmenu event is sent to an element when the right button of the mouse is clicked on it, but before the context menu is displayed. In case the context menu key is pressed, the event is triggered on the html element. Any HTML element can receive this event. right click on the jQuery image to see what contextmenu() can do.
